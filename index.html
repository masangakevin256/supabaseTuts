Supabase Methods

Client Initialization
createClient()

Initialize the Supabase client with your project URL and anon key.

import { createClient } from '@supabase/supabase-js'

const supabaseUrl = 'YOUR_SUPABASE_URL'
const supabaseKey = 'YOUR_SUPABASE_ANON_KEY'

const supabase = createClient(supabaseUrl, supabaseKey)

export default supabase;
Select Method
.select()

Fetch data from a table. Can be chained with other methods.

const { data, error } = await supabase
  .from('smoothies')
  .select()
  .order('created_at', { ascending: false });

if (error) throw error;
return data;
Insert Method
.insert()

Insert new records into a table.

const { data, error } = await supabase
  .from('smoothies')
  .insert([{
    title: 'Berry Blast',
    method: 'Blend all ingredients',
    rating: 5
  }]);

if (error) throw error;
console.log('Inserted:', data);
Update Method
.update()

Update existing records. Usually combined with .eq() to specify which record.

const { data, error } = await supabase
  .from('smoothies')
  .update({
    rating: 4,
    method: 'Updated method description'
  })
  .eq('id', id)
  .select();

if (error) throw error;
return data;
Delete Method
.delete()

Delete records from a table. Always use with .eq() to avoid deleting all records!

const { data, error } = await supabase
  .from('smoothies')
  .delete()
  .eq('id', smoothieId);

if (error) throw error;
console.log('Deleted:', data);
Filtering Methods
.eq() .single()

Filter results and fetch single records.

// Filter by equality
const { data, error } = await supabase
  .from('smoothies')
  .select()
  .eq('id', id)
  .single(); // Get single record

// Other filtering methods available:
// .neq(), .gt(), .lt(), .gte(), .lte()
// .like(), .ilike(), .is(), .in()
Order Method
.order()

Sort results by a column in ascending or descending order.

// Order by created_at descending
const { data, error } = await supabase
  .from('smoothies')
  .select()
  .order('created_at', {
    ascending: false
  });

// Order by title ascending
const { data: sortedData } = await supabase
  .from('smoothies')
  .select()
  .order('title'); // ascending: true by default
Complete CRUD Example

Here's a complete example showing all methods working together:

// 1. Initialize client
const supabase = createClient(url, key);

// 2. CREATE - Insert new record
async function createSmoothie() {
  const { data } = await supabase
    .from('smoothies')
    .insert([{ title, method, rating }]);
  return data;
}

// 3. READ - Fetch all records
async function getSmoothies() {
  const { data } = await supabase
    .from('smoothies')
    .select()
    .order('created_at', { ascending: false });
  return data;
}

// 4. UPDATE - Update a record
async function updateSmoothie(id, updates) {
  const { data } = await supabase
    .from('smoothies')
    .update(updates)
    .eq('id', id);
  return data;
}

// 5. DELETE - Delete a record
async function deleteSmoothie(id) {
  const { data } = await supabase
    .from('smoothies')
    .delete()
    .eq('id', id);
  return data;
}
